# Sam Fortunato's Portfolio

This is the GitHub repository for Sam Fortunato's online portfolio, without the image files.

The website was designed by myself.

## Live Website

You can view the live website at [samfortunato.com](http://samfortunato.com).

## Info

This website was designed by me to be a personal portfolio to showcase some work I've done. It includes examples of graphic design, video, and web work, some info about myself, and links to personal GitHub and LinkedIn pages.

It also hosts a blog with content written by myself. Viewable at the live site.

The site was designed to be responsive, taking a standard mobile-first approach to designing the website. It has been tested at multiple different screen sizes.

I've also used the site to experiment with a few things, such as having markup that defines icons for a touch screen (viewable if you add a shortcut to samfortunato.com to your phone's home screen), markup that defines the window color on Android phones (it's set to a close-to-black color), and usage of the "role" attribute on certain tags for accessibility reasons.

The website's CSS was written in plain CSS initially, but was converted to Sass (SCSS syntax) later on.
