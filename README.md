# Sam Fortunato's Portfolio

Personal portfolio for myself, designed by me. :)

## Live Website

You can view the live website at [samfortunato.com](http://samfortunato.com).

## Info

This website was designed by me to be a personal portfolio to showcase software development work that I've done.

The site was designed to be responsive, taking a standard mobile-first approach to designing the website. It has been tested at multiple different screen sizes.

I've also implemented features such as icons for mobile home screen bookmarks, social media share cards, and a tab "theme color" for Chrome on mobile.

This website utilizes HTML5, and Sass.
